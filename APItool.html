APItool documentation for APItool

Section 0 (struct): APItool_Function_docs

documentation for an APItool_Function.

fields:
char* description
what the function does
char* returns
notes specifically on the return value of the function
char** argsarray_docs
documentation for each argument of the function

Section 1 (struct): APItool_Function

a function token in an APItool AST.

fields:
char* funcname
the name of the function
char* returntype
the type it returns
int argsarray_len
how many arguments the function takes
char** argsarray_names
the names of all of the arguments of the function
char** argsarray_types
the types of all the arguments of the function
[APItool_Function_docs](Section 0)* doc
documentation, if there is any

Section 2 (struct): APItool_Variable

a variable in an APItool AST.

fields:
char* varname
the name of the variable
char* type
the variable's type
char* doc_description
the documentation for the variable

Section 3 (enum): APItool_typedef_status

how a struct, union, or enum is declared

values:
APItool_TYPEDEF
just a typedef (example: typedef struct { ... } NAME;)
APItool_IMMEDIATE
just a name (example: struct NAME { ... };)
APItool_TYPEDEF_IMMEDIATE
both a typedef and name (example: typedef struct NAME { ... } NAME;)

Section 4 (struct): APItool_Struct_Union_Enum_docs

documentation for either an APItool_Struct_Union or an APItool_Enum.

fields:
char* description
its description
char** typesarray_docs
documentation for either the values (enum) or the fields (struct/union).

Section 5 (struct): APItool_Struct_Union

either a struct or a union in an APItool AST.

fields:
char* sunname
its name
int typesarray_len
how many items there are
char** typesarray_names
the name of each item
char** typesarray_types
the type of each item
[APItool_typedef_status](Section 3) marked
how it's declared
[APItool_Struct_Union_Enum_docs](Section 4)* doc
its documentation
bool isunion

Section 6 (struct): APItool_Enum

an enum in an APItool AST.

fields:
char* enumname
its name
int array_len
how many values it has
char** array_names
each value
[APItool_typedef_status](Section 3) marked
how it's declared
[APItool_Struct_Union_Enum_docs](Section 4)* doc
its documentation

Section 7 (union): APItool_token_value

all the different values a token in an APItool AST can have.

fields:
char* verbatim
exact text to be copied into the generated header file.
[APItool_Function](Section 1)* func
a function
[APItool_Variable](Section 2)* var
a variable
[APItool_Struct_Union](Section 5)* sun
a struct or union
[APItool_Enum](Section 6)* enm
an enum

Section 8 (enum): APItool_token_type

see APItool_token_value.

values:
APItool_FUNCTION
APItool_VERBATIM
APItool_VARIABLE
APItool_STRUCT_UNION
APItool_ENUM

Section 9 (struct): APItool_token

an APItool token in an AST.

fields:
[APItool_token_value](Section 7) token
the token's value (contains a pointer)
[APItool_token_type](Section 8) tt
the token's type (enum)

Section 10 (struct): APItool_AST

an APItool AST.

fields:
[APItool_token](Section 9)** val
all of the stored tokens.
int numtokens
how many tokens there are.

Section 11 (function): APItool_gen_pair

generate a header (.h) and documentation (.html) file from an APItool AST.

returns void

args:
[APItool_AST](Section 10)* src
the AST.
char* headerdest
the name of the header.
char* htmldest
the name of the html file.
char* progname
the name of the program.

Section 12 (function): APItool_parse_file

generate an APItool AST from a .h.apitool file.

returns [APItool_AST](Section 10)*

args:
char* fname
the name of the .h.apitool file.

Section 13 (function): APItool_freeAST

free an APItool AST.

returns void

args:
[APItool_AST](Section 10)* a
the AST to free.
